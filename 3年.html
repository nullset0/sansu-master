➗ わり算

💡 わり算は 九九のうらがえし！ 15 ÷ 3 = ？ → 「3 × 何 = 15？」と考えて → 3×5=15 だから 5

わり算の意味（2つある）

① 等分除（同じ数ずつ 分ける）
れい：12個のあめを 3人で分ける → 1人 何個？ → 12÷3=4個

② 包含除（何人に わたせる？）
れい：12個のあめを 1人に4個ずつ → 何人に わたせる？ → 12÷4=3人

あまりのある わり算

あまり＝割り切れずに残る数
れい：14 ÷ 3 → 3×4=12、14−12=2 → 「4 あまり 2」
あまりは「わる数」より小さい。

0 のわり算

0 ÷ 何でも = 0（なにもないものは 分けても なにもない）
⚠️ 何 ÷ 0 は できない（0で分けることは ない）

わり算の文章題

「分ける」「同じ数ずつ」「何人で」 → わり算！
れい：「24本の えんぴつを 3人で 同じだけ。1人 何本？」
→ 24÷3 = 8本

⭕ 円と球（きゅう）

💡 円は どんなかたち？ まんなかの点（中心）から 同じ長さの ところを つないだ かたち。

円の3つのことば

① 中心（ちゅうしん）：円のまんなかの点
② 半径（はんけい）：中心から円のふちまで
③ 直径（ちょっけい）：円を まっすぐ つきぬける線（中心を通る）
★ 直径 = 半径 × 2！
れい：半径 5cm → 直径 10cm
れい：直径 8cm → 半径 4cm

コンパスの 使いかた

▸ コンパスの はりを 中心に さす
▸ えんぴつの ほうを 半径の 長さに 開く
▸ クルッと 1まわり → 円の 完成！

例題

問題：あめが 84こ ある。3人で 同じだけ 分ける。1人 何こ？
→ 「分ける」「同じだけ」 → わり算！
→ 84 ÷ 3 = 28こ
問題：1ふくろ12こ入りの あめ を 7ふくろ買った。ぜんぶで 何こ？
→ 「○こずつ△ふくろ」 → かけ算！
→ 12 × 7 = 84こ

2ステップの問題

問題：1個150円のジュースを 4本 買って、1000円札で払うと、おつりは？
① 4本ぶんの値段：150 × 4 = 600円
② おつり：1000 − 600 = 400円

🧠 3年生は「分かれ目」の学年

  • わり算・小数・分数という新概念が一気に増えます。九九が完璧なら9割解決。
  • わり算で詰まる子は、九九が不安定なケースが多いです。2年の九九問題に戻るのも有効。
  • 小数と分数は「同じものの違う表現」。0.5 = 1/2 を体感させると後で楽。
  • 「あまりのあるわり算」は概念を絵で描いて確認すると定着します。

📅 おすすめの使い方

  • 毎日5分。寝る前30分前後がベスト。
  • 料理や買い物で「半分（1/2）」「3つに分けて（1/3）」と日常に分数を持ち込むと理解が深まります。
